** Starting Forth

Exercises from [[https://www.forth.com/starting-forth/][Starting Forth by Leo Brodie]].

*** CHAPTER 1: Fundamental Forth

**** c1p1

  #+begin_src forth
    : GIFT ." Bookends" ;
    : GIVER ." Bob" ;
    : THANKS CR ." Dear " GIVER ." , thanks for the " GIFT  ." ." ;
    THANKS
  #+end_src

**** c1p2

 #+begin_src forth
   : TEN-LESS ( n -- n )   10 - ;
   assert1( 100 TEN-LESS 90 = ) \ true (-1).
 #+end_src
